# Let's Chat
  - A real-time chat application using the following platforms
    - Front-End: React JS, Socket.io.Client
    - Back-End: Node JS, Socket.io
    - Database: MongoDB
    
## Run the project:
   - Run **npm install** and **npm start** on both server and client side
